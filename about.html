Behind the Scenes: Latest Upgrades & Smarter Workflows

I’ve always believed that good engineers solve problems — but great ones eliminate them entirely. Over the past few months, I've shifted deeper into the world of automation, developing my own tools to supercharge BIM workflows using Python, Dynamo, and the Revit API.

What started as a few scripts to ease repetitive tasks evolved into a toolkit of custom solutions — from filtering Revit elements with intuitive UI forms, to automatically renaming views, and even managing pyRevit extensions like a pro.

But lately, there’s a new tool in my toolbox — AI.

Whether it’s optimizing design decisions, generating smarter code snippets, or accelerating documentation, I’ve begun exploring how artificial intelligence can amplify creativity and reduce friction in BIM workflows. This is just the beginning — and I’m excited to push the boundaries even further.

I’m not just modeling buildings anymore. I’m building tools — and now, I’m teaching them to think smarter too.

  • BIMBuddy Tools: Custom pyRevit buttons with branded forms (filtering, renaming, inspecting)
  • CloneBuddy: Smart extension manager that clones GitHub tools and fixes them automatically
  • HT UI Branding: Consistent interface design across all tools with embedded logos & styles
  • AI + Revit Automation: Using AI to assist in Python scripting, tool design, and logic generation
  • GitHub Portfolio Site: Launched my own digital hub with polished layout & project showcase
