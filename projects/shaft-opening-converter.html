Visualize Shaft Openings (Dynamo Tool)

v1.0.0 · Last updated: July 2025

Visualize and compare shaft openings using DirectShape solids from both host and linked Revit models.

Summary: Shaft Opening Converter is a Dynamo-based toolset that generates visible 3D solids from otherwise invisible "shaft openings" in Revit. It supports both the active project and any linked models. These solids are useful for visualization, model coordination, and exporting to Navisworks.

Key Features:

  • 🔁 Two scripts included: one for host models, another for linked files
  • 🧱 Converts Revit shaft openings into 3D DirectShape solids
  • 🎯 Helps compare architectural vs structural model differences
  • 👁 Viewable in Revit and exported NWC files (Navisworks)
